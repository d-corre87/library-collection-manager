# Library Collection Manager

A frontend web application for managing a small library collection.

This project was built as part of a frontend refresher using HTML, CSS, and JavaScript. It is themed around a law library workflow and includes both manual record management and Open Library API integration.

## Features

- Add new library records
- Edit existing records
- Delete records with confirmation
- Search local collection records
- Sort records A-Z and Z-A
- Filter by format: Books, Journals, and eBooks
- Save records using localStorage
- Search the Open Library API
- Display Open Library search results
- Import selected Open Library records into the local collection
- Responsive card-based layout

## Technologies Used

- HTML5
- CSS3
- JavaScript
- localStorage
- Open Library Search API

## Notes

This is a demo application. Data is stored locally in the user's browser using localStorage and is not shared with other users or saved to a server.
